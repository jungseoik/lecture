# K-Study Match — Design System

> 외국인 유학생을 위한 **한국 대학 비교·추천·상담 플랫폼**의 디자인 시스템.
> 신뢰감 있는 에듀/핀테크 SaaS 룩 — 로열 블루를 중심으로, 깨끗한 화이트와 부드러운
> 블루-그레이 표면, 그리고 파스텔 아이콘 칩이 친근함을 더합니다.

---

## 1. 제품 컨텍스트

K-Study Match(영문 가칭 *Study Korea Match*)는 한국 유학을 희망하는 외국인 학생이
한국 대학의 **등록금·장학금·기숙사·지역 생활비·외국인학생 비율·희망학과**를 쉽게 비교하고,
본인의 학업·어학·재정 조건에 맞는 대학을 추천받은 뒤 특정 학교에 **유학 상담을 요청**할 수
있도록 돕는 웹 플랫폼입니다.

핵심 UX 흐름: **정보 제공 → 신뢰 형성 → 맞춤 추천 → 학생 정보 입력 → 대학 추천 →
학교별 생활비 계산 → 유학 상담요청**. 첫 화면에서는 개인정보 입력을 요구하지 않고
실질적 가치(생활비·장학금·등록금 비교)를 먼저 보여주는 것이 원칙입니다.

사용자용 웹사이트는 **3개 핵심 페이지**로 단순화됩니다:

1. **메인페이지** (`/`) — 서비스 소개, 간단 생활비 확인, 맞춤 추천 유도
2. **맞춤 학교 찾기** (`/match`) — 단계형 학생 정보 입력 → 맞춤 대학 추천 → 학교별 생활비 계산 → 상담요청
3. **전체 대학 보기** (`/universities`) — 대학 검색·필터·정렬, 카드 리스트, 상세 비교

회원가입·로그인·마이페이지·상세보기·상담요청은 흐름을 방해하지 않도록 **모달/보조 화면**으로 연결됩니다.
관리자 페이지는 별도 CRM 구조(이 디자인 시스템의 범위 밖, 향후 확장).

### 출처 자료 (Sources)
이 디자인 시스템은 다음 첨부 자료를 근거로 제작되었습니다. *(독자가 접근 권한을 가졌다고 가정하지 않으나, 추후 참조를 위해 기록)*
- `개발 기획서.pdf` — 프로젝트 개요, 페이지별 기능, DB 구조 초안, MVP 범위

## 2. CONTENT FUNDAMENTALS — 카피 톤 & 보이스

- **언어:** 1차는 **한국어**(영어 확장 예정). 모든 UI 카피는 한국어 기준.
- **존댓말 / 권유체:** 사용자를 부드럽게 이끄는 **해요체 명령·권유형**.
  예) "내 조건에 맞는 학교를 **찾아보세요**", "월 생활비를 **확인하세요**", "상담을 **요청하세요**".
- **불안 해소 → 행동 유도:** 헤드라인은 학생의 막연함을 짚고("한국 유학, **막연하게 준비하지 마세요.**")
  곧바로 구체적 행동을 제시합니다("등록금, 장학금, 기숙사, 생활비를 비교하고 내 조건에 맞는 한국 대학을 찾아보세요.").
- **질문형 섹션 제목:** "이 사이트에서 무엇을 **확인할 수 있나요?**", "한국 유학 가면 한 달 생활비가 얼마나 **들까?**"
  — 학생이 실제로 궁금해하는 말투를 그대로 사용.
- **숫자는 정직하고 구체적으로:** "예상 월 생활비는 약 **650,000원**입니다." 통화는 항상 `원`, 천 단위 콤마.
- **주의/면책 문구는 작고 차분하게:** "추천 결과는 입력한 정보와 대학별 등록 정보를 기준으로 제공됩니다.
  최종 입학 가능 여부와 장학금 수혜 여부는 각 대학의 심사 결과에 따라 달라질 수 있습니다."
- **버튼 라벨은 동사 + 목적:** "내 조건에 맞는 학교 찾기", "무료로 맞춤 학교 확인하기", "유학 상담요청",
  "내 조건으로 학교 추천받기". 1차 MVP에서 상담 버튼명은 **"유학 상담요청"** 으로 통일.
- **신뢰 키워드:** *무료, 한눈에, 비교, 맞춤, 내 조건.* "무료 프로필 만들기"처럼 부담을 낮추는 표현을 헤더 CTA에 사용.
- **이모지 미사용.** 의미 전달은 파스텔 아이콘 칩으로. 영문 약어(GPA, TOPIK, IELTS, TOEFL)는 대문자 그대로.

## 3. VISUAL FOUNDATIONS — 비주얼 기반

**컬러 무드.** 단일 **로열 블루**(`--brand` = `#1F5BDB`)가 모든 1차 액션, 링크, 핵심 숫자(가격·점수·비율)를
담당합니다. 배경은 화이트와 아주 옅은 블루-그레이(`--neutral-50`), 히어로는 부드러운 하늘색 그라데이션
(`--bg-hero` → 화이트). 보라/오렌지/그린/스카이의 **파스텔 페어**는 오직 아이콘 칩·일러스트 배지에만 쓰여
개념을 구분합니다(생활비=그린, 장학금=퍼플, 등록금=블루, 기숙사=오렌지, 상담=스카이). 큰 면적의 그라데이션이나
보라색 배경은 사용하지 않습니다.

**타이포그래피.** Pretendard 단일 패밀리. 헤딩은 **무겁고(700–800) 타이트**(letter-spacing −0.02~−0.03em),
본문은 차분한 400~500, 줄간격 1.6 안팎. 숫자는 같은 패밀리의 700 + `tabular-nums`(`.ksm-tnum`)로 정렬.
디스플레이 46–48px, H1 36px, H2 28px, 본문 15px, 캡션 13px. 1920 기준이 아닌 ~1120px 컨테이너 웹 스케일.

**배경 & 표면.** 풀블리드 사진 대신 **카드 기반 레이아웃**. 페이지 배경은 화이트, 섹션 구분은 옅은 블루
배경 박스(`--blue-50`)로. 히어로 일러스트(캠퍼스+학생)는 외부 이미지 에셋이며, 그 위에 **플로팅 화이트 정보
카드**(등록금/장학금/기숙사/생활비)가 떠 있는 것이 시그니처 모티프입니다.

**카드.** 화이트 표면, **14px 라운드**, 1px 헤어라인 보더(`--border-subtle`), 낮고 차가운 그림자(`--shadow-sm`).
호버 시 살짝 떠오르며(`translateY(-2px)`) 그림자가 깊어집니다(`--shadow-lg`). 패널/히어로 박스는 18px 라운드.

**버튼 & 인터랙션.** 1차=블루 솔리드, 2차=화이트+블루 보더, subtle=옅은 블루 필. 라운드 10px.
- *호버:* 1차는 더 진한 블루(`--brand-hover`), 2차/ghost는 옅은 블루/그레이 배경.
- *프레스:* `translateY(1px)`로 살짝 눌림(스케일 축소 아님).
- *포커스:* 3px 블루 글로우 링(`--shadow-focus`).

**폼 컨트롤.** 인풋/셀렉트는 8px 라운드, 헤어라인 보더, 포커스 시 블루 보더 + 글로우. 셀렉트는 커스텀
쉐브론. 체크박스는 블루 스퀘어(5px 라운드) + 화이트 체크, 라디오는 블루 닷. 멀티스텝 폼은 번호 원형
스테퍼(완료=그린 체크, 현재=블루 필, 예정=뮤트).

**배지 & 태그.** 작은 6px 라운드 필. `solid`(블루, 추천도 점수·핵심 마커)는 아껴 쓰고, 매칭 사유는
그린(`희망전공 일치`), 조건 충족은 옅은 블루(`기숙사 제공`, `TOPIK 조건 충족`), 학교 유형은 뉴트럴(`국립대`).

**모션.** 빠르고 절제됨 — 120ms(빠름)~200ms(보통), `ease-standard`. 페이드 + 미세한 상승. 바운스/무한
루프 없음. 시안 톤은 따뜻한 햇살(벚꽃) 일러스트와 차분한 블루 UI의 조합.

**라운드 스케일.** 칩 6 · 인풋 8 · 버튼 10 · 카드 14 · 패널 18 · pill. **그림자**는 xs~xl 모두 차가운
블루-그레이(`rgba(22,28,43,…)`), 낮고 퍼지는 형태. 투명/블러는 sticky 헤더(`backdrop-filter: blur(8px)`)에만.

## 4. ICONOGRAPHY — 아이코노그래피

- **아이콘 세트:** 코드베이스에 자체 아이콘 에셋이 제공되지 않아, 시안의 **둥글고 채워진(fill) 듀오톤**
  스타일에 가장 가까운 **[Phosphor Icons](https://phosphoricons.com/)** 를 CDN으로 채택했습니다(`@phosphor-icons/web@2.1.1`).
  → *대체(substitution) 사실을 명시합니다. 확정 아이콘 세트가 있으면 알려주세요.*
- **사용 규칙:** 기능/개념 아이콘은 **파스텔 아이콘 칩(`IconChip`)** 안에 `ph-fill` 가중치로 넣습니다.
  툴바·인라인 액션(북마크, 펼치기, 화살표, 검색)은 칩 없이 `ph`(regular) 가중치를 직접 사용.
- **개념 ↔ 컬러 매핑:** 생활비 `wallet`(green) · 장학금 `graduation-cap`(purple) · 등록금 `currency-krw`(blue) ·
  기숙사 `buildings`(orange) · 상담 `chat-circle-dots`(sky) · 외국인 비율 `users-three`(sky) · 감면율 `seal-percent`(purple).
- **로고:** 별도 심볼 없는 **워드마크** — "K-Study"(브랜드 블루) + "Match"(잉크/700). `components/core/Logo.jsx` 참조.
- **이모지/유니코드 아이콘:** 사용하지 않음. 모든 픽토그램은 Phosphor.
- 로드: `<link rel="stylesheet" href="https://unpkg.com/@phosphor-icons/web@2.1.1/src/regular/style.css">` 와
  동일 경로의 `/src/fill/style.css`.

## 5. INDEX — 파일 안내 (매니페스트)

**글로벌 진입점**
- `styles.css` — 소비자가 링크하는 단일 파일(`@import`만 나열).

**토큰** (`tokens/`)
- `colors.css` — 블루/뉴트럴 스케일, 파스텔 페어, 시맨틱, 의미 별칭
- `typography.css` — 폰트 스택, 가중치, 타입 스케일, 자간
- `spacing.css` — 4px 그리드, 레이아웃 별칭(컨테이너/섹션)
- `effects.css` — 라운드, 그림자, 모션
- `fonts.css` — Pretendard `@font-face`(CDN)
- `base.css` — 리셋 + 문서 기본 타이포

**컴포넌트** (`components/`)
- `core/` — `Button`, `IconButton`, `Logo`
- `forms/` — `Input`, `Select`, `Checkbox`, `Radio`/`RadioGroup`, `FormField`, `StepIndicator`
- `data-display/` — `Card`, `Badge`, `IconChip`, `StatCard`, `FeatureCard`
- `feedback/` — `Modal`(로그인·회원가입·상담요청·확인 다이얼로그)
- 각 디렉터리에 `*.card.html`(Design System 탭 미리보기) 포함. 사용: `const { X } = window.KStudyMatchDesignSystem_624b2d`.

**UI 킷** (`ui_kits/`)
- `marketing-website/` — 메인 / 맞춤 학교 찾기 / 전체 대학 보기 3개 화면 (인터랙티브 `index.html`). 로그인·회원가입·유학 상담요청 모달(`Modals.jsx`) 연결. 자체 `README.md` 참조.

**파운데이션 specimen 카드** (`guidelines/`)
- 타입(`type-*`), 컬러(`colors-*`), 스페이싱/라운드/그림자(`spacing-scale`,`radius`,`shadows`), 브랜드(`brand-logo`).

**기타**
- `SKILL.md` — Agent Skill 호환 진입점.
- `HANDOFF.md` — 개발 핸드오프(화면→컴포넌트→필드→DB→API 매핑, 추천 점수·생활비 산식).
- `_ds_bundle.js` / `_ds_manifest.json` / `_adherence.oxlintrc.json` — 컴파일러 자동 생성(직접 편집 금지).
